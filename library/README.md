## Limitations

### Change detection

Supporting change detection in reversible systems is not planned currently. This is postponed until Bevy being more stable as the available approach is dependent on numerous implementation details of Bevy. See the relevant [tracking issue](https://github.com/Spotshade/bevy-reversible-systems/issues/2).
